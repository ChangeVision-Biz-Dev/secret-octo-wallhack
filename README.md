# SimpleDiagram
JavascriptのCanvasを利用してお絵かきDiagramを作る。  
まずは○、△、□が描けること。

## 技術
Canvas、イベントがらみは（お勉強もかねて）Javascriptをそのまま使うことを前提とする。  
ただし、デファクトスタンダードなライブラリがあればは使う（momentjs等）。

## 設計
MVCかMVVMか？
* MVVM  
 * 自力でのMVVMはつらいかも。

## マイルストーン
何か動くものができたらいいね。
* Rect、Circle、Line、Path等。
* テキスト、カラー等のスタイル。
* 移動、拡大縮小、回転。
* トランザクション、Undo、Redo。
* コピペ。
